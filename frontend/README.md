# Easy GE Finder Frontend

This React application fetches course rankings from a backend API and lists the top General Education classes by average grade.

## Development

1. Install dependencies
   ```bash
   npm install
   ```
2. Set the API endpoint in `.env`
   ```
   VITE_API_URL=http://localhost:9000/easy-ge
   ```
3. Run the development server
   ```bash
   npm run dev
   ```
